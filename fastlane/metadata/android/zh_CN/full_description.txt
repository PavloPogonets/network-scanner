Network Scanner 是一款快速、轻量且注重隐私的工具，用于发现和分析本地网络中的设备。

<b>主要功能：</b>

• 快速网络扫描，发现 WiFi 网络上的所有设备
• 详细的设备信息，包括 IP 地址、MAC 地址和主机名
• 端口扫描，识别开放端口和运行中的服务
• 设备类型检测与分类
• Material Design 3 界面，现代直观的 UI
• 一键刷新，快速重新扫描
• 可自定义的扫描设置

<b>隐私优先：</b>

• 无广告或追踪
• 无分析或数据收集
• 无需外部互联网连接——所有流量均留在本地网络中
• 所有扫描均在设备本地完成
• 仅申请必要权限
• 100% 开源（GPL-3.0）

<b>适用场景：</b>

• 网络管理员监控家庭/办公网络
• 技术爱好者探索自己的网络环境
• 排查网络连接问题
• 发现网络中的未授权设备
• 查找网络设备（打印机、摄像头、智能家居设备）的 IP 地址

<b>权限说明：</b>

Network Scanner 仅申请网络扫描所必需的权限：
• 网络访问 - 与网络上的设备通信
• WiFi 状态 - 检查是否已连接到 WiFi
• 位置（不用于追踪）- Android 对 WiFi 扫描的要求

Network Scanner 完全无广告、无追踪器，并尊重您的隐私。所有代码均为开源且可审查。
